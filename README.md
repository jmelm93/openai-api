## Requirements for Text Embedding/Related Link Recommender
1. CSV w/ columns " " and " " stored within `data` directory
2. `.env` file with the below:
   1. `CONTENT_CSV_PATH` (e.g., "filename" - should not include .csv at end)
   2. `OPENAI_API_KEY` (string secret key for OpenAI API)
3. Download the requirements.txt into a virtual env
4. Start the virtual env
5. run `python main.py`
